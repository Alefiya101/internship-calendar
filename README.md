# Internship Calendar Platform

A community-driven, calendar-based platform to help students track verified internship openings and deadlines.

## Problem
Students often miss internship opportunities because listings are scattered across platforms, shared late, or buried in noisy feeds.

## Solution
A centralized calendar that displays internship openings and deadlines, allowing community submissions with admin moderation.

## MVP Features
- Monthly calendar view of internships
- Internship cards with deadlines and official links
- Community submissions
- Admin approval workflow
- Automatic expiry of closed internships

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: SQL
